# Ejercicio 06 - Login con QtDesigner

## 📌 Descripción

En este ejercicio se replica una aplicación de **login** utilizando **QtDesigner** para la creación de la interfaz gráfica.

El objetivo es comprender el flujo de trabajo basado en diseño visual, integrando archivos `.ui` con código en C++.

---

## 🎯 Objetivos

* Diseñar la interfaz utilizando QtDesigner
* Integrar el archivo `.ui` con la lógica en C++
* Manejar eventos de usuario (login)
* Comprender la separación entre diseño y lógica

---

## 🧩 Tecnologías utilizadas

* C++
* Qt (Widgets + QtDesigner)
