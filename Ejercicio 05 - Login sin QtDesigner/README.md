# Ejercicio 05 - Login sin QtDesigner

## 📌 Descripción

En este ejercicio se replica una aplicación básica de **login** utilizando **Qt**, partiendo desde un **empty project**, sin utilizar QtDesigner.

El objetivo es comprender la construcción manual de aplicaciones QT mediante código, aplicando conceptos de Programación Orientada a Objetos y el uso de layouts, widgets y señales/slots.

---

## 🎯 Objetivos

* Crear una interfaz de login desde cero
* Utilizar layouts para organizar los componentes
* Manejar eventos mediante señales y slots
* Aplicar buenas prácticas en la estructura del código

---

## 🧩 Tecnologías utilizadas

* C++
* Qt (Widgets)
